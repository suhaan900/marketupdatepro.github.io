Files included:
- index.html : responsive main page (mobile + desktop)
- config.js  : put your Alpha Vantage API key here
- main.js    : fetches live quotes and draws the chart (Chart.js)

How to use:
1. Get a FREE API key from Alpha Vantage: https://www.alphavantage.co/support/#api-key
2. Edit config.js and replace YOUR_API_KEY_HERE with your key.
3. Unzip and upload all files to a GitHub repository named 'marketupdatepro' (or any name).
4. In GitHub: Settings -> Pages -> Branch: main / (root) -> Save
5. Visit: https://<your-github-username>.github.io/<repo-name>

Notes:
- Alpha Vantage free tier allows 5 requests per minute; the script includes delays to respect limits.
- If you want faster updates or more symbols, consider a paid API.
